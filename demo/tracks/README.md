# tracks using for demo or AR 

## Content

Each directory contains three files: 
- accel.csv : content of accelerometer
- gryro.csv : content of gyroscop 
- calib.csv : a csv to synchronize accel.csv and gryro.csv files. the content of sensor_second_start video_frame_start represent the same time on each file
- movie.MOV : the track recording (use to build track map)
